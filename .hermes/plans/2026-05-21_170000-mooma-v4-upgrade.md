## Step 1 — Product Data Schema (`src/data/products.json`)

Create centralized JSON data store. One file, all products.

```
src/
├── data/
│   └── products.json   ← single source of truth
```

**Schema fields (NO price fields):**
- `slug` (string)
- `title` (string)
- `category` (string)
- `emoji` (string)
- `badge` (string)
- `badgeClass` ("badge-hot" | "badge-new" | "badge-rec")
- `suitable` (string)
- `highlight` (string)
- `note` (string)
- `stars` (string, e.g. "★★★★★ 4.8")
- `image` (string | null)
- `shopeeUrl` (string)
- `keywords` (string[])
- `metaTitle` (string)
- `metaDescription` (string)
- `featured` (boolean)
- `priority` (number 1-10)
- `lastUpdated` (ISO date string)

**Hard ban:** No `price`, `discountPrice`, `salePrice`, `currency`, `promotionPrice`, `originalPrice`.

## Step 2 — ProductCard Component Upgrade

Add `variant` prop to `src/components/ProductCard.astro`:

```
variant: 'homepage' | 'category' | 'product' | 'compare' | 'article'
```

Variant behaviors:
- **homepage** — full card with image, badge, meta rows, 2 buttons
- **category** — compact card, emoji-only, badge, title, 1 CTA
- **product** — hero layout, full gallery, specs grid
- **compare** — side-by-side layout, highlights only
- **article** — inline mention with CTA button

All variants:
- NO price display
- `rel="sponsored noopener noreferrer"` on Shopee links
- Affiliate disclosure near CTA

## Step 3 — No-Price Scanner (`scripts/no-price-check.sh`)

Pre-build scanner that greps all source files for banned terms:

Script checks `src/` for:
```
฿, บาท, ราคา, ลดราคา, โปรโมชัน, ส่งฟรี, ถูกสุด, เหลือเพียง, 
flash sale, discount, sale price, ฿[0-9], [0-9] บาท
```

Exit code 1 = found banned term → build fails.
Integrated into `package.json` → `"prebuild": "bash scripts/no-price-check.sh"`

## Step 4 — Affiliate Disclosure Check (`scripts/check-disclosure.sh`)

## Step 10 — Build & Deploy Checklist (`scripts/deploy.sh`)

Single deploy script:
```bash
npm run check          # Astro typecheck
bash scripts/no-price-check.sh
bash scripts/check-disclosure.sh
npm run build          # Astro build
# verify key URLs
curl -sI https://mooma.online/ | grep "200"
curl -sI https://mooma.online/sitemap.xml | grep "200"
# deploy
wrangler pages deploy dist/ --project-name=mooma-online
```

`package.json`:
```json
"scripts": {
  "check": "astro check",
  "validate": "bash scripts/no-price-check.sh && bash scripts/check-disclosure.sh",
  "prebuild": "npm run validate",
  "build": "astro build",
  "deploy": "bash scripts/deploy.sh"
}
```
